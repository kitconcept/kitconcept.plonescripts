============
Installation
============


Stable release
--------------

To install kitconcept.scripts, run this command in your terminal:

.. code-block:: console

    $ pip install kitconcept.scripts

This is the preferred method to install kitconcept.scripts, as it
will always install the most recent stable release.

If you don't have `pip <https://pip.pypa.io>`_ installed, this
`Python installation guide <https://docs.python-guide.org/starting/installation/>`_
can guide you through the process.

From sources
------------

The sources for kitconcept.scripts can be downloaded from the
`Github repo <https://github.com/kitconcept/kitconcept.scripts>`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/kitconcept/kitconcept.scripts

Or download the `tarball <https://github.com/kitconcept/kitconcept.scripts/tarball/master>`_:

.. code-block:: console

    $ curl  -OL https://github.com/kitconcept/kitconcept.scripts/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install
